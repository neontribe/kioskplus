Allowed domain: The domain of the site that is being walled in, e.g. archivealive.org
Elements to remove: A comma-separated list of jQuery element selectors that will be removed from each page, e.g. .socials, .register
 
Timeout after: The period of inactivity, in seconds, after which to show the timeout screen
Warning period: The length of time, in seconds, to show the timeout warning screen for
Warning message: Warning message displayed to user before engaging slideshow; [tp] will be replace by the timeout value converted to minutes, [wp] will be replaced by the warning period value
Number of slides: The total number of slides, each of which must be named number.jpg (e.g. 1.jpg, 2.jpg, etc.) and placed in the extension's "images/slides/" folder
Slide delay: The number of seconds to show each slide for
Transition duration: The length in seconds of the fade transitions
Slideshow tagline: The message displayed over the slides
Slides: Slides are stored in a folder called "images/slides/" inside the extension's folder. This folder is chrome_profile_path/Extensions/extension_id. You can find your profile path on the Chrome version page (chrome://version/), and the extension ID listed on the Extensions page.
